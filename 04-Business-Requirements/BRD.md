# Business Requirements Document (BRD)

## Purpose
The purpose of this project is to analyze employee attrition patterns and provide actionable business recommendations to improve retention.

---

# Business Requirements

| ID | Business Requirement |
|----|----------------------|
| BR-01 | Analyze overall employee attrition rate |
| BR-02 | Identify departments with highest attrition |
| BR-03 | Analyze impact of salary on attrition |
| BR-04 | Evaluate work-life balance influence |
| BR-05 | Measure relationship between overtime and attrition |
| BR-06 | Analyze job satisfaction levels |
| BR-07 | Identify high-risk employee demographics |
| BR-08 | Provide interactive filtering capabilities |
| BR-09 | Generate strategic retention recommendations |

---

# Functional Requirements

| ID | Functional Requirement |
|----|------------------------|
| FR-01 | Dashboard shall display KPI metrics |
| FR-02 | Dashboard shall include interactive slicers |
| FR-03 | Dashboard shall visualize attrition trends |
| FR-04 | Dashboard shall support department analysis |
| FR-05 | Dashboard shall support salary analysis |
| FR-06 | Dashboard shall display employee demographics |

---

# Non-Functional Requirements

| ID | Non-Functional Requirement |
|----|-----------------------------|
| NFR-01 | Dashboard should load quickly |
| NFR-02 | Visuals should be easy to understand |
| NFR-03 | Dashboard should be interactive |
| NFR-04 | Dashboard should maintain visual consistency |

---

# Expected Outcomes

- Improved understanding of employee turnover
- Identification of major attrition drivers
- Data-driven HR decision support
- Strategic workforce recommendations
